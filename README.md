# Monapolis

Forum(?) with Monacoin.

## Demo
heroku? sei0o.cf?

## Features
- Give and Take Monacoin
- Threads
- Groups(Cities)
- Responses: Post, Delete, Edit, Update
- Up/Down
- User
- Share
- Real-time

## Installation
```
$ git clone https://github.com/sei0o/monapolis.git`
$ cd monapolis
$ rackup
```

## Requirement
- Ruby 2.0+
- Sinatra
